## Intro

这是一个北京航空航天大学“冯如杯”竞赛的 LaTeX 论文模板。如果你是北航学子苦于冯如杯论文格式的复杂，苦于没有 LaTeX 模板，恭喜你发现了本项目。



该项目包含当年冯如杯部分赛道论文的 LaTeX 模板，请尽情使用与分享，如果有使用上的问题请提 issue 或直接联系作者（联系方式见下）。同时也欢迎更多同学参与到本项目的维护中来。



官方模板格式要求见 document 内 .doc 文件。
